Expand description

Dispatches one round’s worth of edges. Production: spawn one nix copy subprocess per edge via std::thread. Sim: madsim’s task model + deterministic time.

Implementations must run all edges concurrently (no serial fallback) — the cascade’s whole point is in-round parallelism.
