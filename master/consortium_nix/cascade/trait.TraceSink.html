Expand description

Receives RoundSnapshots as the cascade runs. Default impls in crate::cascade_trace include a Vec-collecting recorder + JSON / DOT / ASCII exporters.

Implementations are called from inside the cascade coordinator loop so they should be cheap. Heavy work (rendering, network IO) belongs downstream — collect snapshots, render after the run completes.
