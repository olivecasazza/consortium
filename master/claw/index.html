claw — parallel cluster command execution (replaces clush).

Usage: claw -w node[1-5] echo hello # run on nodes claw -w node[1-5] -b ‘uname -r’ # gather identical output claw -w node[1-5] -l root uptime # run as root claw –copy /etc/hosts -w node[1-5] # copy file to nodes echo ‘script’ | claw -w node[1-5] # pipe stdin to command
