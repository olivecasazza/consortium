pub struct TreeWorker { /* private fields */ }
Expand description

Worker that distributes command execution across a cluster topology.

TreeWorker uses PropagationTreeRouter to determine routing through gateway nodes, and PropagationChannel to communicate with gateways. Nodes directly reachable (same hop as root) are executed via direct workers (SSH or local).

Mirrors ClusterShell’s TreeWorker class.
