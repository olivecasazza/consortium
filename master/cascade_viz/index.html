cascade-viz — visualize cascade event streams.

Subcommands: replay <TRACE_FILE> Replay a JSONL trace file and render live Run a fresh scenario and render

Renderers + sinks come from consortium_cli::event_render. The binary is the thin shell that picks an EventSink (collector or JsonlWriter) for the live path, parses JSONL for the replay path, and delegates everything else to event_render::render_events.
