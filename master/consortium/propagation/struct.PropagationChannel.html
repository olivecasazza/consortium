Expand description

Admin-node propagation channel.

Manages the lifecycle of communication with a gateway node:

  1. Init: Send configuration (serialized topology).
  2. Config: Wait for ACK, then send control message (command).
  3. Control: Wait for ACK, then enter gather state.
  4. Gather: Collect results (stdout, stderr, retcodes, timeouts, routing).

This is a state machine that drives the Channel transport.
