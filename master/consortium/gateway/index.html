Gateway node logic.

Rust implementation of ClusterShell.Gateway.

A gateway is a tree node that relays commands between the admin node and the subtree below it. This module contains the target (non-initiator) side of a tree channel:

  • GatewayChannel: high-level logic for gateways — accepts the channel greeting, receives the topology configuration, ACKs it, and converts processing failures into Error replies followed by channel close (mirrors Gateway.GatewayChannel).
  • TreeWorkerResponder: builds routing notifications for the gateway channel with the corrected RoutingMessage construction of upstream commit 490323d.
