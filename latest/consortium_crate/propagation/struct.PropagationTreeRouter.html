pub struct PropagationTreeRouter { /* private fields */ }
Expand description

Performs route resolving operations within a propagation tree.

The router parses the topology tree to generate a routing table, then provides next_hop() and dispatch() methods for directing messages to the correct gateway.

Load balancing is achieved by tracking the number of connections (fanin) per gateway node.

pub fn dispatch(&self, dst: &NodeSet) -> Vec<(NodeSet, NodeSet)>

Dispatch nodes from a target nodeset to directly connected gateways.

Acts as an iterator, yielding (gateway_nodeset, target_nodes) pairs. Provides load balancing between gateways. Remaining nodes that don’t match any route are treated as directly connected.
