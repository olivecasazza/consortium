Task orchestration.

Rust implementation of ClusterShell.Task.

A Task manages a coordinated collection of independent parallel Worker objects, providing a high-level API for executing commands locally or across remote nodes and collecting their results.

§Example

use consortium::task::Task;

let mut task = Task::new();
task.shell("hostname", None, None);
task.run(None).unwrap();

for (rc, nodes) in task.iter_retcodes(None) {
    println!("rc={}: {:?}", rc, nodes);
}
